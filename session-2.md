جزوه آموزشی جلسه کلاس برنامه نویسی پایتون
این جزوه آموزشی بر اساس متن تبدیل شده از کلاس برنامه نویسی پایتون تهیه شده است و شامل جزئیات کامل، مثال های کد پایتون و توضیحات علمی جامع می باشد.

مقدمه ای بر برنامه نویسی و حل مسئله
در برنامه نویسی، هدف اصلی حل مسائل است. معمولاً مسائل بزرگ را به قطعات کوچکتر تقسیم می کنیم. سپس هر قطعه کوچکتر را به صورت جداگانه حل کرده و در نهایت با ترکیب راه حل ها به نتیجه مطلوب می رسیم. این فرایند به ویژه در پروژه های بزرگ برنامه نویسی کاربرد دارد.

محیط توسعه یکپارچه (IDE)
برای نوشتن کد برنامه نویسی، به یک محیط کد نویسی نیاز داریم که به آن محیط توسعه یکپارچه یا IDE گفته می شود.  زمانی که پایتون را نصب می کنید، یک IDE ابتدایی به نام IDLE به همراه آن نصب می شود.

IDLE (Integrated Development and Learning Environment) یک محیط کد نویسی پایه است که امکان نوشتن و اجرای کدهای پایتون را فراهم می کند.  علاوه بر IDLE، IDE های پیشرفته تری مانند Spyder, PyCharm, Visual Studio Code و غیره نیز وجود دارند که امکانات بیشتر و حرفه ای تری را ارائه می دهند.

ویژگی های IDE ها:

ویرایشگر کد (Code Editor): محل نوشتن کد برنامه نویسی با امکاناتی مانند رنگ بندی کلمات کلیدی، تکمیل خودکار کد و ... که به خوانایی و سهولت کد نویسی کمک می کند.
اجرا کننده کد (Code Executor/Interpreter): امکانی برای اجرای کدهای نوشته شده و مشاهده خروجی برنامه.
خطایاب (Debugger): ابزارهایی برای یافتن و رفع خطاها در کد برنامه.
پایتون یک زبان مفسری

پایتون یک زبان برنامه نویسی مفسری (Interpreted) است.  برخلاف زبان های کامپایلری (Compiled) که کد منبع ابتدا به کد ماشین تبدیل شده و سپس اجرا می شود، در زبان های مفسری، کد خط به خط توسط مفسر (Interpreter) در زمان اجرا ترجمه و اجرا می گردد.

مفسر پایتون مسئول اجرای کدهایی است که شما نوشته اید.  هنگامی که یک برنامه پایتون را اجرا می کنید، مفسر پایتون خط به خط کد را می خواند، ترجمه می کند و اجرا می نماید.

زبان های رسمی (فرمال) و محاوره ای
در دنیای زبان ها دو دسته بندی کلی وجود دارد:

زبان های رسمی یا فرمال (Formal Languages): این زبان ها دارای قواعد و دستورات بسیار دقیق و سختگیرانه ای هستند. زبان های برنامه نویسی مانند پایتون، جاوا، سی++ و غیره در این دسته قرار می گیرند. برای نوشتن کد به زبان های فرمال، باید قوانین زبان را به دقت رعایت کرد، در غیر این صورت با خطا مواجه خواهیم شد.
زبان های محاوره ای (Conversational Languages): این زبان ها زبان هایی هستند که در مکالمات روزمره استفاده می کنیم. قواعد گرامری در این زبان ها انعطاف پذیرتر است و ممکن است در صحبت روزمره بسیاری از قواعد دستوری را رعایت نکنیم، بدون اینکه در انتقال مفهوم مشکلی ایجاد شود.
پایتون یک زبان برنامه نویسی رسمی است.  بنابراین، برای نوشتن کدهای صحیح در پایتون، باید قوانین و ساختار (Syntax) این زبان را به دقت یاد بگیریم و رعایت کنیم. عدم رعایت قوانین زبان منجر به خطاهای نحوی (Syntax Errors) خواهد شد.

خطاهای نحوی (Syntax Errors)
خطاهای نحوی زمانی رخ می دهند که قوانین ساختاری زبان برنامه نویسی نقض شوند.  به عنوان مثال، اشتباه تایپی در دستورات زبان، جا انداختن پرانتز، استفاده نادرست از عملگرها و غیره می توانند منجر به خطاهای نحوی شوند.

مثال خطا نحوی:

Python

printt("Hello") # خطای نحوی - دستور print به صورت اشتباه تایپ شده است
در مثال بالا، به دلیل اشتباه تایپی در دستور print (به صورت printt نوشته شده)، مفسر پایتون قادر به تشخیص دستور نیست و خطای نحوی (SyntaxError) صادر می کند.

استفاده از پایتون به عنوان ماشین حساب
یکی از ساده ترین راه ها برای شروع کار با پایتون، استفاده از آن به عنوان یک ماشین حساب است. محیط کنسول پایتون (در IDLE یا Spyder) به شما امکان می دهد عبارات ریاضی را مستقیماً وارد کرده و نتیجه را مشاهده کنید.

مثال های استفاده از پایتون به عنوان ماشین حساب:

Python

>>> 5 + 5
10
>>> 8 * 2
16
>>> 10 / 2
5.0
>>> 2 ** 3  # توان (2 به توان 3)
8
تابع print() برای نمایش خروجی
برای نمایش متن یا نتایج محاسبات در پایتون، از تابع print() استفاده می شود.  تابع print() مقداری را که به عنوان ورودی (آرگومان) به آن داده می شود، در خروجی چاپ می کند.

ساختار تابع print():

Python

print(مقدار_برای_چاپ)
مثال های استفاده از تابع print():

Python

print("Hello, world!") # چاپ متن "Hello, world!"
print(5 + 5) # چاپ نتیجه محاسبه 5 + 5 (عدد 10)
در نسخه های قدیمی تر پایتون، تابع print بدون پرانتز استفاده می شد، اما در نسخه های جدید، استفاده از پرانتز الزامی است.

عملگرها (Operators)
عملگرها نمادهایی هستند که برای انجام عملیات مختلف بر روی مقادیر (عملوندها) به کار می روند.  پایتون از عملگرهای متنوعی پشتیبانی می کند، از جمله عملگرهای ریاضی (Arithmetic Operators).

عملگرهای ریاضی در پایتون:

عملگر	نام عملگر	مثال
+	جمع	5 + 2 = 7
-	تفریق	8 - 3 = 5
*	ضرب	4 * 6 = 24
/	تقسیم	9 / 3 = 3.0
**	توان	2** 4 = 16

Export to Sheets
مثال های عملی با عملگرهای ریاضی:

Python

>>> 2 + 8
10
>>> 9 - 2
7
>>> 3 * 2
6
>>> 2 ** 3
8
نکته در مورد عملگر توان (**):  عملگر توان (**) برای محاسبه توان به کار می رود.  برای مثال، 2 ** 3 یعنی 2 به توان 3. نباید بین دو علامت ستاره (**) فاصله وجود داشته باشد.  در غیر این صورت با خطای نحوی (SyntaxError) مواجه خواهید شد.

مثال خطای نحوی در عملگر توان:

Python

>>> 2 * * 3 # خطای نحوی - فاصله بین دو علامت ستاره در عملگر توان
SyntaxError: invalid syntax
مقادیر و انواع داده (Data Types)
در برنامه نویسی، مقدار (Value) به داده ای گفته می شود که برنامه با آن کار می کند.  هر مقدار در پایتون دارای یک نوع داده (Data Type) است که دسته بندی آن را مشخص می کند.

انواع داده پایه در پایتون:

اعداد صحیح (Integers - int): اعداد بدون ممیز اعشاری مانند 4، 10، -5 و غیره.
اعداد اعشاری (Floating-point numbers - float): اعداد دارای ممیز اعشاری مانند 8.5، 3.14، -2.7 و غیره.
رشته ها (Strings - str): دنباله ای از کاراکترها که برای نمایش متن استفاده می شوند. رشته ها در پایتون درون علامت نقل قول (quotation marks) قرار می گیرند (می توان از نقل قول تکی ('...') یا دوتایی ("...") استفاده کرد). مانند "Hello", 'Python', "123" و غیره.
تابع type() برای تشخیص نوع داده

برای تشخیص نوع داده یک مقدار در پایتون، از تابع type() استفاده می شود.  تابع type() نوع داده مقدار ورودی خود را برمی گرداند.

ساختار تابع type():

Python

type(مقدار)
مثال های استفاده از تابع type():

Python

>>> type(5) # نوع داده عدد 5
<class 'int'> # int مخفف integer (عدد صحیح)
>>> type(8.5) # نوع داده عدد 8.5
<class 'float'> # float مخفف floating-point number (عدد اعشاری)
>>> type("Hello") # نوع داده رشته "Hello"
<class 'str'> # str مخفف string (رشته)
>>> type('Python') # نوع داده رشته 'Python'
<class 'str'>
>>> type("123") # نوع داده رشته "123" (به دلیل وجود علامت نقل قول)
<class 'str'>
>>> type(123) # نوع داده عدد صحیح 123 (بدون علامت نقل قول)
<class 'int'>
دیباگینگ و انواع خطاها
دیباگینگ (Debugging) فرایند یافتن و رفع خطاها (Bugs) در کد برنامه نویسی است.  خطاها در برنامه نویسی معمول هستند و بخشی جدایی ناپذیر از فرایند توسعه نرم افزار به شمار می روند.

انواع خطاها در برنامه نویسی:

خطاهای نحوی (Syntax Errors):  همانطور که قبلاً گفته شد، این خطاها زمانی رخ می دهند که قوانین ساختاری زبان برنامه نویسی نقض شوند.  IDE ها معمولاً به شناسایی این خطاها در حین کد نویسی کمک می کنند.

خطاهای زمان اجرا (Runtime Errors):  این خطاها در زمان اجرای برنامه (Runtime) رخ می دهند.  به عنوان مثال، تلاش برای تقسیم بر صفر، تلاش برای باز کردن فایلی که وجود ندارد، دسترسی به اندیس خارج از محدوده یک لیست و غیره می توانند منجر به خطاهای زمان اجرا شوند.  برنامه ممکن است تا نقطه ای اجرا شود و سپس به دلیل خطای زمان اجرا متوقف (Crash) گردد.

مثال خطای زمان اجرا (تقسیم بر صفر):

Python

>>> 10 / 0
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
ZeroDivisionError: division by zero
خطاهای منطقی (Logical Errors):  این خطاها پیچیده ترین نوع خطاها هستند.  در این نوع خطا، کد برنامه بدون هیچ خطا نحوی یا زمان اجرا اجرا می شود و خروجی تولید می کند.  اما خروجی تولید شده با نتیجه مورد انتظار متفاوت است.  به عبارت دیگر، منطق برنامه اشتباه است و برنامه آنطور که باید کار نمی کند.  یافتن و رفع خطاهای منطقی معمولاً زمان برترین و دشوارترین بخش دیباگینگ است. برای دیباگ کردن خطاهای منطقی، برنامه نویس باید قدم به قدم کد را دنبال کند و متغیرها و پردازش های انجام شده را بررسی کند تا منشاء خطا را پیدا کند.

متغیرها و عملگر تخصیص
متغیرها (Variables) نام هایی هستند که به یک مکان در حافظه اشاره می کنند.  متغیرها برای ذخیره سازی مقادیر در برنامه استفاده می شوند.  به هر متغیر می توان یک مقدار اختصاص داد.

عملگر تخصیص (=) برای اختصاص دادن مقدار به یک متغیر استفاده می شود.  علامت مساوی (=) در پایتون به معنای برابری ریاضی نیست، بلکه به معنای تخصیص است.

ساختار عملگر تخصیص:

Python

نام_متغیر = مقدار
مثال های تخصیص مقدار به متغیر:

Python

message = "Hello, Python!" # تخصیص رشته "Hello, Python!" به متغیر message
number = 10 # تخصیص عدد صحیح 10 به متغیر number
در مثال بالا، متغیر message ایجاد شده و رشته "Hello, Python!" در آن ذخیره شده است.  همچنین متغیر number ایجاد شده و عدد صحیح 10 در آن ذخیره شده است.

قوانین نامگذاری متغیرها:

شروع با عدد مجاز نیست: نام متغیر نمی تواند با عدد شروع شود، اما می تواند شامل عدد در وسط یا انتهای نام باشد. مثال: variable1 (مجاز)، 1variable (غیر مجاز).
استفاده از کاراکترهای خاص غیر مجاز است: نام متغیر نمی تواند شامل کاراکترهای خاص مانند @, #, $, %, ^, &, *, (, ), -, +, =, !, ~ به جز کاراکتر Underscore (_) باشد.
کلمات کلیدی رزرو شده غیر مجاز است: نام متغیر نمی تواند از کلمات کلیدی رزرو شده (Reserved Keywords) زبان پایتون باشد. کلمات کلیدی کلماتی هستند که در زبان پایتون دارای معنی خاصی هستند و برای اهداف خاصی رزرو شده اند (مانند class, def, if, else, for, while, True, False, None و غیره). IDE ها معمولاً کلمات کلیدی رزرو شده را با رنگ متفاوتی نمایش می دهند.
استفاده از Underscore برای فاصله گذاری: برای نامگذاری متغیرهای چند کلمه ای، از Underscore (_) برای جدا کردن کلمات استفاده می شود. به این روش Snake Case گفته می شود. مثال: first_name, user_age, calculate_area.
خوانا و معنادار بودن نام متغیر: انتخاب نام های خوانا و معنادار برای متغیرها بسیار مهم است. نام متغیر باید به گونه ای باشد که هدف و محتوای متغیر را به خوبی نشان دهد. این امر به خوانایی و نگهداری بهتر کد کمک می کند.
مثال های نامگذاری صحیح و غلط متغیرها:

نام متغیر	وضعیت	توضیح
my_variable	صحیح	استفاده از Snake Case
variable1	صحیح	شامل عدد در انتها
_count	صحیح	شروع با Underscore
FirstName	صحیح	استفاده از Camel Case
1st_variable	غلط	شروع با عدد
my-variable	غلط	استفاده از کاراکتر - (غیر مجاز)
my variable	غلط	استفاده از Space (غیر مجاز)
class	غلط	کلمه کلیدی رزرو شده

Export to Sheets
مثال های عملی با متغیرها و عملگر تخصیص:

Python

>>> message = "Welcome to Python!"
>>> message # نمایش مقدار متغیر message
'Welcome to Python!'
>>> type(message) # نمایش نوع داده متغیر message

>>> number_of_students = 25

انواع داده ایستا (Static) و پویا (Dynamic)
در بحث نوع داده (Data Type) زبان های برنامه نویسی به دو دسته ایستا (Static) و پویا (Dynamic) تقسیم می شوند:

زبان های ایستا (Statically Typed Languages): در زبان های ایستا، نوع داده متغیر در زمان تعریف (کامپایل) مشخص می شود و قابل تغییر نیست.  برنامه نویس باید به صراحت نوع داده متغیر را هنگام تعریف آن مشخص کند.  زبان هایی مانند جاوا، سی++، سی شارپ از نوع داده ایستا استفاده می کنند.

مثال (زبان های ایستا):

Java

String name = "Ali"; // تعریف متغیر name از نوع String در جاوا
int age = 30;      // تعریف متغیر age از نوع int در جاوا
زبان های پویا (Dynamically Typed Languages): در زبان های پویا، نوع داده متغیر در زمان اجرا (Runtime) تعیین می شود و قابل تغییر است.  برنامه نویس نیازی به مشخص کردن صریح نوع داده متغیر ندارد.  نوع داده متغیر به صورت خودکار بر اساس مقداری که به آن اختصاص داده می شود، تشخیص داده می شود.  زبان هایی مانند پایتون، جاوااسکریپت، روبی از نوع داده پویا استفاده می کنند.

مثال (زبان های پویا - پایتون):

Python

name = "Ali" # نوع داده متغیر name به طور خودکار String تشخیص داده می شود
age = 30   # نوع داده متغیر age به طور خودکار int تشخیص داده می شود
مزیت زبان های پویا مانند پایتون:

سهولت و سرعت توسعه: عدم نیاز به تعریف صریح نوع داده، کد نویسی را سریع تر و ساده تر می کند.
انعطاف پذیری بیشتر: امکان تغییر نوع داده متغیر در طول برنامه، انعطاف پذیری برنامه را افزایش می دهد.
ادامه مباحث در جلسات آینده
در جلسات آینده، مباحث پیشرفته تری مانند کتابخانه ها (Libraries)، توابع (Functions)، ساختارهای کنترلی (Control Structures) و برنامه نویسی شی گرا (Object-Oriented Programming) در پایتون بررسی خواهند شد و نوشتن کدهای پیچیده تر و کاربردی تر آغاز خواهد شد.
